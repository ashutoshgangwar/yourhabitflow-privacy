# YourHabit Flow - Privacy Policy

This repository hosts the privacy policy for the YourHabit Flow mobile application.

## Accessing the Privacy Policy

The privacy policy is available online at:

**https://ashutoshgangwar.github.io/yourhabitflow-privacy/**

## Setup

This privacy policy is hosted using GitHub Pages. To enable GitHub Pages for this repository:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select the branch (usually `main`)
4. Click Save

The privacy policy will be available at the URL above once GitHub Pages is enabled.

## Local Development

To view the privacy policy locally:

```bash
# Clone the repository
git clone https://github.com/ashutoshgangwar/yourhabitflow-privacy.git

# Open index.html in your browser
open index.html
```
